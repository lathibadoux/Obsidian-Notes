## 007.1.1 Navigation and Directory Management
|Command|Description|
|---|---|
|`pwd`|Show your current directory (“print working directory”).|
|`ls`|List files and folders in the current directory.|
|`ls -l`|Detailed list (shows permissions, size, and dates).|
|`ls -a`|Show _all_ files (including hidden ones like `.env`).|
|`cd folder_name`|Move into a folder.|
|`cd ..`|Move up one folder level.|
|`cd /`|Move to the root directory.|
|`cd ~`|Move to your home directory.|
|`mkdir folder_name`|Create a new folder.|
|`rmdir folder_name`|Delete an _empty_ folder.|
|`rm -r folder_name`|Delete a folder _and its contents_ (be careful).|
|`touch file_name.py`|Create a new empty file.|
|`rm file_name.py`|Delete a file.|
|`clear`|Clear the terminal window.|

## 007.1.2 Working with Files
|Command|Description|
|---|---|
|`cat file_name`|Print the contents of a file.|
|`less file_name`|View file contents one page at a time.|
|`cp source destination`|Copy a file or folder.|
|`mv source destination`|Move or rename a file/folder.|
|`echo "text" > file.txt`|Write text to a file (overwrites).|
|`echo "text" >> file.txt`|Append text to a file.|
|`code .`|Open the current folder in VS Code.|

## 007.1.3 Python & Virtual Environments
|Command|Description|
|---|---|
|`python --version`|Check your Python version.|
|`pip list`|Show installed Python packages.|
|`pip install package_name`|Install a package.|
|`pip install -r requirements.txt`|Install from a requirements file.|
|`pip freeze > requirements.txt`|Create a requirements file from your environment.|
|`python -m venv venv`|Create a new virtual environment in a folder named `venv`.|
|`source venv/Scripts/activate`|Activate the venv (Git Bash).|
|`.\venv\Scripts\Activate.ps1`|Activate the venv (PowerShell).|
|`deactivate`|Deactivate the venv.|
|`python file_name.py`|Run a Python script.|

## 007.1.4 Git and GitHub Basics
| Command                                              | Description                                                                   |
| ---------------------------------------------------- | ----------------------------------------------------------------------------- |
| `git init`                                           | Start a new Git repository.                                                   |
| `git status`                                         | Check which files are staged/unstaged.                                        |
| `git add .`                                          | Stage all changes.                                                            |
| `git commit -m "message"`                            | Commit changes with a message.                                                |
| `git branch`                                         | Show current branch.                                                          |
| `git checkout main`                                  | Switch to the main branch.                                                    |
| `git checkout branch_name`                           | Switch to another branch.                                                     |
| `git checkout -b branch_name`                        | Create a new branch and switch to it.                                         |
| `git merge branch_name`                              | Merge a branch into the current one.                                          |
| `git remote -v`                                      | Show connected remote repos.                                                  |
| `git push origin main`                               | Push your code to GitHub.                                                     |
| `git push -u origin main`                            | Set origin as upstream                                                        |
| `git pull`                                           | Get the latest changes from GitHub.                                           |
| `git clone <repo_url>`                               | Download (clone) a repository.                                                |
| `git stash push -u -m "WIP before pulling (Oct 22)"` | Stash all local chgit staanges (including untracked changes)                  |
| `git fetch origin`                                   | Updates local view from remote repository, but doesn't change any local files |

## 007.1.5 Web Server & FastAPI Commands
|Command|Description|
|---|---|
|`uvicorn main:app --reload --port 8000`|Run your FastAPI app locally (dev mode).|
|`uvicorn main:app --host 0.0.0.0 --port $PORT`|Run your FastAPI app on Render (prod mode).|
|`curl http://localhost:8000`|Test a local API endpoint from the terminal.|
|`curl http://localhost:8000/docs`|Open the FastAPI docs programmatically.|
## 007.1.6 System & Utility Commands
| Command                 | Description                        |
| ----------------------- | ---------------------------------- |
| `whoami`                | Show your username.                |
| `date`                  | Display the current date and time. |
| `history`               | Show recent terminal commands.     |
| `grep "text" file_name` | Search for text in a file.         |
| `head file_name`        | Show the first 10 lines of a file. |
| `tail file_name`        | Show the last 10 lines of a file.  |
| `exit`                  | Close the terminal session.        |
