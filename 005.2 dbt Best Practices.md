1) Create separate folders in the 'models' folder for staging tables/views and marts.

![[2025-10-01 12_39_05-Greenshot.jpg]]

2) Set the default materialization for staging tables to 'view'.  Set the default materialization for marts as 'table'.

**In  the dbt_project.yml file:**
```sql
models:

  jaffle_shop:

    staging:

      +materialized: view

    marts:  

      +materialized: table
```
