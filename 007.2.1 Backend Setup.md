### 4Ô∏è‚É£ Create a `.env` file for secrets (API keys, etc.)

In the project root, create `.env`:
```text
OPENAI_API_KEY=sk-...your real key here...
```

Rules:
- No quotes
- No spaces around `=`
- This file should be at the **same level as `.venv`**

Also add it to `.gitignore`:
==Prevents API keys, passwords, credentials, or environment variables accidentally pushed to GitHub.==
```text
# .gitignore
.env
.venv/
```

### 5Ô∏è‚É£ Create your main Python file

Example: `app.py` or `limerick.py` in the project root:
```python
from dotenv import load_dotenv
import os
from openai import OpenAI

# Load environment variables from .env
load_dotenv()

api_key = os.getenv("OPENAI_API_KEY")
client = OpenAI(api_key=api_key)

completion = client.chat.completions.create(
    model="gpt-4o",
    messages=[
        {"role": "system", "content": "You're a helpful assistant."},
        {"role": "user", "content": "Write a limerick about Python."},
    ],
)

print(completion.choices[0].message.content)
```

(You can also just do `client = OpenAI()` if `OPENAI_API_KEY` is set in the environment.)
